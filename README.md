# ftms-bike
Turn a Raspberry Pi into a Bluetooth Low Energy (BLE) cycling sensor. Uses a hall effect sensor to detect crank revolutions and broadcasts cadence, power, and speed to fitness apps like Zwift, MyWhoosh, and Kinomap via the FTMS (Fitness Machine Service) standard. Tested on Raspberry Pi Bullseye, with Kinomap and MyWhoosh confirmed working well.
